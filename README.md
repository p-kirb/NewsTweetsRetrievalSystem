# NewsTweetsRetrievalSystem
Repository for a search engine that retrieves breaking news tweets from the official accounts of CNN, BBC, and The Economist between 2010-2021.


### Folder Structure:
```
Project
├── datasets
│   ├── tweets_bbc.csv
│   ├── tweets_cnn.csv
│   └── tweets_eco.csv
└── NewsTweetsRetrievalSystem
    └── build_index.py
```
